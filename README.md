# 📊 Sales Analytics Dashboard

## Overview
This project is an interactive Power BI dashboard built using the SuperStore Sales dataset. It provides insights into sales performance, profitability, regional trends, and product performance through interactive visualizations.

## Dashboard Preview

![Sales Dashboard](Dashboard_Screenshot.png)

## Key KPIs
- Total Sales
- Total Orders
- Total Profit
- Profit Margin

## Visualizations
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Top 10 Products by Sales
- Profit by Segment
- Sales by State

## Filters
- Region
- Category
- Segment
- Year

## Tools Used
- Power BI
- Power Query
- DAX

## Key Insights
- Technology generated the highest sales.
- Profit margin is approximately 11.6%.
- Sales performance varies significantly across regions and states.
- Interactive slicers allow dynamic business analysis.
