# 📊 Sales Analytics Dashboard

## Overview
This project is an interactive Power BI dashboard built using the SuperStore Sales dataset. It provides insights into sales performance, profitability, regional performance, and product trends.

- Region
- Category
- Segment
- Year

## Tools Used
- Power BI
- Power Query
- DAX

## Key Insights
- Technology generated the highest sales.
- Profit margin is approximately 11.6%.
- Sales performance varies significantly across regions and states.
- Interactive slicers allow dynamic business analysis.
